# ProConnect — Documentación de Casos de Uso

Esta carpeta contiene la documentación en Markdown de los casos de uso de ProConnect, organizada por módulos funcionales.

La documentación fue consolidada a partir de lo trabajado en el proyecto: autenticación, perfiles, servicios, reservas, paquetes, pagos, emails, colas, reseñas, Docker y pendientes técnicos como LiveKit, WebSockets, CI/CD y despliegue.
